# Description

This is a simple library to work with the LCD Alphanumeric Display, founded in many sizes, the most commons are 16x2 and 20x4.

This lib aiming to provide this advantages:

- Allow use any amount of displays you want;
- Easy port no another chipsets, with any direct link with the microcontroller
- Easy to use and configure
